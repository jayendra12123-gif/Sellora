# Sellora Admin

Admin dashboard built with React and Vite.

## Requirements

- Node.js 18+
- npm 9+

## Run

From `admin/`:

```bash
npm install
npm run dev
```

App runs at `http://localhost:5174` by default (or the next available port).

## Scripts

- `npm run dev`
- `npm run build`
- `npm run preview`
